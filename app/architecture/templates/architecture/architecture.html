COMMENTS :
  • Cleaning and the Modeling were done upstream on jupyter notebook in order to feed a database hosted on an external server listing the access path of the created models.
  • User connects to the webapp hosted on a server to enter his mushroom identification criteria and choose a prediction model.
  • Application connects to the database to retrieve and load the model and performs the user's prediction.
  • Application is deployed in a continuous way, the whole code being centralized on a GITHUB repository.
